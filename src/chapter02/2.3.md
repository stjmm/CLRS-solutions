# 3-3
Initial state:
At the start of iteration we have
Subarray A[p:k-1] contains k - p smallest elements of L and R
L[i] and R[i] are the smallest elements of their respective arrays that have not yet been copied to A

Maintance:
- Loop compares L[i] and R[i]
- Picks smaller and puts in A[k]
- Since L[i] and R[j] were the smallest remaining, and A[p…k−1] already contained all smaller elements, A[p…k] remains sorted and contains the k−p+1 smallest elements.
- Incrementing k and i or j maintains the invariant for the next round

Termination:
When loop finishes one of L or R is exhausted. The remaining elements in one of them are by definition larger than all elemens already in A[p:k-1]. The cleanup simply puts those elements in remaining space

# 3-4

Proof by induction:

**Base case**
Let $n = 2^1 = 2$.
From our equation $T(2) = 2$.
From our proposed formula $T(n) = n \lg n$: $T(2) = 2 \lg 2$, we have $T(n) = 2$
So base case holds.

**Induction hypothesis**
Because n is a power of 2, we assume that $n = 2^k$. Assume:
$$T(2^k) = 2^k \lg (2^k)$$

**Inductive step**
We need to show that its true for $n = 2^{k+1}$.
$$T(2^{k+1}) = 2T\left(\frac{2^{k+1}}{2}\right) + 2^{k+1}$$

Simplify
$$T(2^{k+1}) = 2T(2^k) + 2^{k+1}$$

Substitute induction hypothesis
$$T(2^{k+1}) = 2(2^k \lg (2^k)) + 2^{k+1}$$

Using lg 2^k = k
$$T(2^{k+1}) = 2(2^k \cdot k) + 2(2^{k+1})
$$T(2^{k+1}) = 2^{k+1} \cdot k + 2(2^{k+1})

Factor out $2^{k+1}$
$$T(2^{k+1}) = 2^{k+1}(k + 1)$$

Since $k + 1 = \lg(2^{k+1})$
$$T(2^{k+1}) = 2^{k+1} \lg(2^{k+1})

Inductive step proven, equal to induction hypothesis

# 3-5

RECURSIVE-INSERTION-SORT(A, n):
    if n == 1
        return
    RECURSIVE-INSERTION-SORT(A, n - 1)

    i = n - 1
    key = A[n]
    while i >= 0 and A[i] > key
        A[i + 1] = A[i]   
        i--
    A[i + 1] = key

Recurrence:
T(1) = O(1)
T(n) = T(n - 1) + O(n)

Unrolling:
$$T(n) = T(n - 1) + O(n) + T(n - 2) + O(n) + ... = ... = 1 + 2 + 3 + 4 + ... + n = O(n^2)$$

RECURSIVE-BINARY-SEARCH(A, p, r, x)
    if p > r
        return false

    q = p + (r - p) // 2
    if A[q] == x
        return true
    else if A[q] > x
        RECURSIVE-BINARY-SEARCH(A, p, q - 1, x)
    else else 
        RECURSIVE-BINARY-SEARCH(A, q + 1, r, x)

ITERATIVE-BINARY-SEARCH(A, n, x)
    p = 0
    r = n - 1
    while (p <= r)
        q = p + (r - p) // 2
        if A[q] == x
            return true
        else if A[q] > x
            r = q - 1
        else if A[q] < x
            p = q + 1

    return false

Each step T(n) = T(n/2) + O(1)
So T(n) = O(log n)
