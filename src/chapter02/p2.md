# b.

**Loop invariant**: At the start of 2-4 for iteration the position of the smallest element of A[i:n] is at most j. This is true on the first iteration because j = n

**Initialization**: At the start the subarray contains A[n] which is the smallest element of the array.

**Maintnance**: j = k, smallest number is at most k. We compare A[k] to A[k - 1], if A[k - 1] is greater then A[k] is smallest element, after swap we know smallest element of A[i:n] must occur at most k - 1 maintaining invariant. If A[k] is greater the smallest element is k -1.

**Termination**: Loop terminates when j is in position i

# c.

**Loop invariant**: At the start of each iteration of the loop in lines 1-4, the subarray A[1:i-1] contains i - 1 smallest elements of A[1..n] in a sorted oreder.

**Initialization**: First loop has 0 elements so its a trivially sorted array.

**Maintnance**: We have subarray A[1:i-1] which contans i - 1 sorted elements. After loop from 2-4 A[i] will be the ith smallest element of the array, and thus A[1:i-1] sorted holds.

**Termination**: Loop terminates with a sorted A[1..n] array
