# CLRS Solutions

My solutions to *Introduction To Algorithms*.

##

Solutions to exercises are in [src](./src/) in their respective chapter.

I also have C++ implementations of many of the books algorithms in [include](./include/).
These are called from .cpp files in src. For example [sorts.cpp](./src/chapter02/sorts.cpp)

### To build use cmake
```bash
$ mkdir -p build
$ cmake -S . -B build
$ cmae --build build
```
